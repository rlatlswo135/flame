# @flame/docs

Documentation site for the flame design system, built with [Next.js 16](https://nextjs.org) and [Tailwind CSS 4](https://tailwindcss.com).

## Getting Started

```bash
pnpm dev  # starts dev server on http://localhost:3000
```

## Tech Stack

- Next.js 16
- Tailwind CSS 4
- `@flame/ui` + `@flame/table` as dependencies
