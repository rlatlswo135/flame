# @flame/ui

Core UI component library for the flame design system.

## Installation

```bash
pnpm add @flame/ui
```

**Peer dependencies:** React 19+, ReactDOM 19+

## Components

| Component | Description |
|-----------|-------------|
| `Dialog` | Modal dialog with compound sub-components (`Trigger`, `Content`, `Closer`) |
| `Funnel` | Step-based navigation (`Step`, `Next`, `Prev`) |
| `Select` | Select input |
| `MultiSelect` | Multi-select input |

For usage examples and interactive demos, see the [Storybook](../../../apps/storybook).

## Development

```bash
pnpm dev    # watch mode
pnpm build  # production build
pnpm test   # run tests
```
