# @flame/table

Table component for the flame design system, built on top of [@tanstack/react-table](https://tanstack.com/table).

## Installation

```bash
pnpm add @flame/table
```

**Peer dependencies:** React 19+, ReactDOM 19+

## Dependencies

- `@tanstack/react-table` — headless table utilities

## Development

```bash
pnpm dev    # watch mode
pnpm build  # production build
pnpm test   # run tests
```
